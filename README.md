# 🚀 ZingBot Cloud API

Official Backend API for ZingBot Cloud.

## 🌍 About

ZingBot Cloud is a SaaS platform that helps Web3 projects create, manage and automate community bots without writing code.

## ✨ Features

- 🔐 User Authentication
- 🏢 Workspace Management
- 🤖 Bot Management
- 📊 Dashboard API
- 💳 Subscription System
- 🤝 Affiliate Program
- 🤖 AI Assistant
- 🎥 Community Meetings

## 🛠️ Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL
- SQLite (Development)
- JWT Authentication
